Introduction

This is a command guide for the bot Unown on Pokémon Showdown!. Unown does not have an auto-invite function. If you would like to add Unown to your room, please contact Kris on Pokémon Showdown!.

etour

  • Usage

    Arguments

    • start - starts the tournament, if there is one
      • Aliases: forcestart
    • end - ends the tournament, if there is one
      • Aliases: forceend
    • name [name] - names the current tournament to whatever [name] is
      • Aliases: setname
    • clearname - resets the tournament's name
      • Aliases: delname
    • autostart [number | "off"] - sets the tournament's autostart timer to the provided number or turns it off if "off" is provided
      • Aliases: setautostart, as
    • autodq [number | "off"] - sets the tournament's autodq timer to the provided number or turns it off is "off" is provided
      • Aliases: setautodq, adq
    • dq [username] - disqualifies [username] from the tournament
      • Aliases: disqualify
    • addrule [rule 1, rule 2, rule 3, rule n] - adds the provided rules to a tournament format; the rules get saved in a database for a tournament, so you can send multiple rules on multiple lines if a single line is too long
    • delrule [rule 1, rule 2, rule 3, rule n] - removes the provided rules from a tournament format
      • Aliases: removerule
    • ban [item 1, Pokémon 1, move 1, ability 1] - bans the provided item/move/ability/Pokémon/tier/etc from a tournament format
      • Aliases: addban
    • unban [item 1, Pokémon 1, move 1, ability 1] - unbans the provided item/move/ability/Pokémon/tier/etc from a tournament format
      • Aliases: removeban
    • clearrules - removes all custom rules from a format
    • viewrules - puts the custom tournament rules on display
    • scout ["on" | "off"] - enables/disables scouting for tournaments
      • Aliases: scouting
    • modjoin ["on" | "off"] - enables/disables allowing users to modjoin their tournament matches; disabled by default
    • cap [number] - sets the tournament's player cap
      • Aliases: playercap
    • [format[, "elim" | "rr"[, player cap[, # of elimination rounds[, name]]]]] - creates a tournament with the following parameters applied; some parameters can be skipped—for example, you can do etour gen8ou, elim,,,Name
  • Permissions: Host, %, @, #, &

What are hosts?

Hosts are people whitelisted via the tourhost command that are allowed to create tours by proxy of Unown for RoA Tour Nights. If you would like to apply to become a host, check out the host signups thread.
